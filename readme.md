# v0.1 AutoCCTV使用文档

## 1. 引言

### 1.1 简述

***AutoCCTV 可在系统时间 18：59 时自动开启默认浏览器并转到 https://tv.cctv.com/live/index.shtml 播放 CCTV13 ，并在开启浏览器5秒后在屏幕中心模拟鼠标双击使直播全屏播放，在系统时间 19：30 时自动关闭浏览器。***

### 1.2 开发环境

Python 3.12.1

### 1.3 仓库地址

https://github.com/Return-Log/AutoCCTV

### 1.4 系统要求

Windows10、Windows11
